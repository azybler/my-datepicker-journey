# Animating the day grid (part 1)

<Row>
  <Column>
    $$
      s_n = \frac{w/c}{w}
    $$

    $$ w = $$ width of the calendar.
    
    $$ c = $$ number of columns, in this case 4.
    
    $$ s_n = $$ the scale we need to transform to.

    ```css
    @keyframes dayGridScaleAnimation {
      0% {
        transform: scale(1);
      }
      100% {
        transform: scale(0.25);
      }
    }

    .day-grid-animate-contract {
      animation: dayGridScaleAnimation 0.3s ease-in forwards;
    }
    ```
  </Column>
